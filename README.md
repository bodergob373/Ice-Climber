# Ice Climber
A Godot game about climbing an ice wall. Use A and D to use arms and click to anchor the ice pick. Avoid rocky areas with thin ice, step onto ledges to regain stamina, and collect crates to heal from falls.

## Controls:

A\left: start using left arm
D\right: start using right arm
Left Click: anchor the hand's ice pick into the ice
Q: quit game and see score
Mouse Movement: move an unanchored arm or body
